**Q: Write a Python program to swap the values of two variables without using a temporary variable.**

```python
# initialization
a = 10
b = 20
print("Before Swapping")

print(f"a :{a}")
print(f"b :{b}")

# Swapping
b,a = a,b

print("After Swapping")
print(f"a :{a}")
print(f"b :{b}")
```
